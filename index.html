Настройка OSPF

Динамическая маршрутизация через FRR + MD5 аутентификация

1
HQ-RTR — Установка FRR и настройка OSPF
▼
HQ-RTR
# Временно включить репозиторий
nano /etc/apt/sources.list
# Добавить:
deb [trusted=yes] http://deb.debian.org/debian buster main

nano /etc/resolv.conf
nameserver 8.8.8.8

apt update
apt install frr -y
nano /etc/frr/daemons
# Изменить:
ospfd=yes

systemctl restart frr
vtysh
conf t
router ospf
network 10.10.10.0/30 area 0
network 192.168.1.0/26 area 0
network 192.168.2.0/28 area 0
network 192.168.3.0/29 area 0
do wr mem
# MD5 аутентификация на туннеле
vtysh
conf t
int gre1
ip ospf authentication message-digest
ip ospf message-digest-key 1 md5 P@ssw0rd
do wr mem
# Отключить временный репозиторий
nano /etc/apt/sources.list
# Закомментировать:
#deb [trusted=yes] http://deb.debian.org/debian buster main
2

Обратный прокси-сервер

Nginx на HQ-RTR для moodle.au-team.irpo и wiki.au-team.irpo

1
HQ-RTR — Nginx Reverse Proxy
▼
HQ-SRV — Обновить config.php Moodle
mcedit /var/www/html/config.php
# Изменить:
$CFG->wwwroot = 'http://moodle.au-team.irpo';
HQ-RTR
apt install nginx -y
nano /etc/nginx/sites-available/proxy
server {
  listen 80;
  server_name moodle.au-team.irpo;
  location / {
    proxy_pass http://192.168.1.2:80;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $remote_addr;
  }
}

server {
  listen 80;
  server_name wiki.au-team.irpo;
  location / {
    proxy_pass http://192.168.4.2:8080;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $remote_addr;
  }
}
rm -rf /etc/nginx/sites-available/default
rm -rf /etc/nginx/sites-enabled/default
ln -s /etc/nginx/sites-available/proxy /etc/nginx/sites-enabled
ls -la /etc/nginx/sites-enabled
systemctl restart nginx
